# Cosmos Scratch

Cosmos Scratch is a research sandbox for world-model and robotics-data
experiments.

## What is inside

- BridgeData inspection notebooks.
- Continuous-tokenizer training scripts and SLURM launch scripts.
- Small checks for overfitting, causality, and experiment plumbing.

## Status

This is scratch research code, not a production Phygularity service. Use it for
experiments that may later inform Envigon or Actigon work.
